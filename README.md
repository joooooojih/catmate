# catmate
펫시터 웹 사이트
> 여행을 가거나 잠시 반려견을 돌봐줄 곳이 필요할 때 펫시터의 집에 반려견을 맡길 수 있게 이어주는 사이트입니다.
* 참고 사이트 : https://www.dogmate.co.kr/ "도그메이트"

# 주 기능
* 예약 시스템
* 실시간 메세지

# 사용한 기술
* 언어 : JAVA, JSP, SpringFramework, Javascript, jquery, AJAX, Mybatis
* OS : Window10
* DBMS : MySql
* 기타 : Apache Server, Tomcat8.5, Maven, Bootstrap
